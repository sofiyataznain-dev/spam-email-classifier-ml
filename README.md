# spam-email-classifier-ml
A machine learning classification project that detects spam messages using NLP and Naive Bayes in Python.
## Tech Stack
- Python
- Pandas
- Scikit-learn
- NLP (TF-IDF)
- Naive Bayes

## How to Run
pip install -r requirements.txt
python spam_classifier.py

## Output
Accuracy score is displayed after model evaluation is
Spam Email Classifier
Accuracy: 0.75
